=== Регистрация в системе ===

Linux -- многопользовательская система. Принцип разграничения пользователей
обеспечивает возможность независимой и безопасной работы нескольких
пользователей на одном компьютере. Зарегистрировавшись, т.е. введя имя (login)
и пароль (password), каждый пользователь приобретает возможность работать в
своём собственном окружении: иметь собственные настройки, получать доступ к
своим данным, закрытым для других пользователей.

Учётная запись пользователя создаётся непосредственно в процессе установки
системы. После загрузки системы всегда можно добавить новых пользователей или
удалить существующих при помощи стандартных средств управления пользователями.

Для регистрации в системе необходимо указать имя пользователя и затем пароль.
После непродолжительного времени ожидания запустится графическая оболочка
операционной системы.

Если систему устанавливали не вы, то имя пользователя и пароль вам должен
сообщить системный администратор, отвечающий за настройку данного компьютера.

[NOTE]

Поскольку работа в системе с использованием учётной записи администратора
небезопасна, вход в систему в графическом режиме для суперпользователя root
запрещён. Попытка зарегистрироваться в системе будет прервана сообщением об
ошибке.
