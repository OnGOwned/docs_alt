= Обзор Xfce 4 =

== Введение ==

Xfce -- лёгкое настольное рабочее окружение, разработанное с целью повышения
производительности. Оно позволяет просто и быстро запускать приложения,
сохраняя ресурсы системы, оставаясь внешне привлекательным и простым в
использовании. 

== Рабочий стол Xfce 4 ==

image::images/desktop-xfce.png[Рабочий стол Xfce 4,align="center"]

На Рабочем столе Xfce 4 есть две особых области. Сверху вниз:

- Область рабочего стола (рабочая площадь в центре, занимающая большую часть
  экрана);
- Панель Xfce (полоса в самом низу).

В левом верхнем углу области рабочего стола расположены значки «Домашний
каталог» и «Корзина».

При двойном щелчке по значку «Домашний каталог» запустится файловый менеджер по
умолчанию с открытым домашним каталогом. При двойном щелчке по значку «Корзина»
запустится файловый менеджер по умолчанию с открытой корзиной.

В домашнем каталоге пользователя хранятся все его файлы. У каждого пользователя
в системе есть свой домашний каталог. По умолчанию, пользователи не могут
получить доступ к домашним каталогам других пользователей.

В корзине расположены удаленные файлы. Чтобы удалить файлы всегда, щелкните
правой кнопкой мыши на корзине и выберите «Очистить корзину».

Чтобы открыть меню рабочего стола, в пустой области щелкните правой кнопкой
мыши и выберите нужный пункт. Например, для изменения фона рабочего стола
выберите «Настройки рабочего стола...».

=== Панель Xfce ===

Панель меню расположена в нижней части экрана.

В левой части панели находятся:

* «Меню Xfce» обеспечивает быстрый доступ ко всем графическим приложениями и
  утилитам установленным на компьютере;
* Кнопка «Скрыть окна и показать рабочий стол» позволяет свернуть все открытые
  окна для быстрого доступа к рабочему столу;
* Набор стандартных значков для запуска программ обеспечивает быстрый доступ к
  браузеру, почтовой программе, терминалу и файловому менеджеру;
* Переключатель рабочих мест позволяет переключаться между доступными рабочими
  местами с помощью мышки.

В правой части находятся:

* Системный лоток отображает значки уведомлений;
* Регулятор громкости -- удобный регулятор громкости для звуковой карты;
* Часы;
* Меню «Переход» предоставляет быстрый доступ к папкам, документам и съёмным
  носителям;
* Кнопка «Завершение сеанса» скрывает в себе завершение сеанса, блокировку
  экрана и другие системные действия.

Между двумя наборами значков расположена область списка задач, в которой
отображаются запущенные приложения.
