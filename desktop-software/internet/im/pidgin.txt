=== Pidgin ===

Pidgin -- клиентская программа -- мультипротокольный мессенджер для обмена
мгновенными сообщениями. Используя только Pidgin, вы можете оставаться в контакте
с привычными адресатами независимо от того, какой протокол или клиентское
ПО они используют. Pidgin позволяет работать с IRC-сетями,
сохранять комментарии к пользователям из контакт‐листа, может
объединять несколько контактов в один мета-контакт.

После запуска Pidgin необходимо произвести его первоначальную настройку.  Диалог
мастера создания учётной записи появляется при первом запуске Pidgin.
Для создания новой учётной записи этот диалог необходимо запустить из меню
*Уч.записи -> Управление учётными записями*.

[NOTE]
Если вы ещё не зарегистрированы ни в одной службе мгновенных сообщений, то
предварительно необходимо зарегистрироваться на соответствующем
веб-сайте.

Из списка поддерживаемых протоколов выберите предпочтительный для использования.
Возможно, вы уже решили, какой протокол IM будете использовать (потому что вы уже
пользовались им, либо потому что им пользуются ваши друзья). Если вы ещё не
решили, выберите протокол, основанный на открытых стандартах, например, XMPP
(также известный, как Jabber).

image::images/pidgin-settings.png[Добавление учётной записи]

После настройки учётной записи добавьте в список контактов ваших собеседников
(кнопка *Добавить собеседника...*) и, при условии, что ваш собеседник подключён
к службе мгновенных сообщений, можете начинать общение. Для этого просто
нажмите на значок, соответствующий определённому контакту, после чего откроется
окно для ввода текста.

За дополнительной информацией по использованию Pidgin можно обратиться к
справке, вызываемой из меню *Помощь -> Помощь в сети*.
