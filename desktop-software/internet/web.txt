== Веб-навигация ==

Для просмотра веб-страниц используются веб-браузеры. Для Linux создано большое
число веб-браузеров. При всех их отличиях, основные принципы работы с любым из
них остаются неизменными. Как правило, веб-браузер предоставляет адресную строку,
куда вносится адрес сайта, который вы желаете посетить. Для более быстрого
доступа к часто посещаемым сайтам их адреса можно добавлять в закладки. Для
перехода к предыдущей/следующей просмотренной веб-странице, как правило,
предусмотрены специальные кнопки на панели инструментов.

Какой именно использовать браузер зависит полностью от ваших предпочтений. Может
быть вы уже знакомы с определённым браузером по опыту работы в других
операционных системах. В любом случае просто попробуйте предлагаемые
веб-браузеры и выберите наиболее удобный для вас.
