== Веб-навигация ==

Для просмотра веб-страниц используются веб-браузеры. Для Linux создано большое
число веб-браузеров. При всех их отличиях, основные принципы работы с любым из
них остаются неизменными. Веб-браузер предоставляет адресную строку в которую
вносится адрес необходимого вам сайта. Для более быстрого доступа адреса часто
посещаемых сайтов можно добавлять в закладки. Для перехода к
предыдущей/следующей просмотренной веб-странице, как правило, предусмотрены
специальные кнопки на панели инструментов.

Выбор браузера полностью зависит от ваших предпочтений. Может быть, по опыту
работы в других операционных системах вы уже знакомы с определённым браузером.
В любом случае, попробуйте предлагаемые веб-браузеры и выберите наиболее
удобный для вас.

include::web/konqueror.txt[Konqueror]
