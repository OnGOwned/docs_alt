=== KMail ===

В качестве программы для создания, отправки и получения электронных писем по
умолчанию используется почтовый клиент KMail.

image::images/kmail.png[Почтовый клиент KMail]

[float]
==== Первоначальная настройка KMail ====

Для того чтобы начать отправлять и получать почту, нужно произвести
первоначальную настройку почтового клиента. При первом запуске KMail это можно
сделать с помощью мастера учётных записей. Последовательно проходя шаги
мастера, выберите тип почтового ящика и введите в предложенные поля сведения об
учётной записи (ваше имя и адрес почтового ящика), её имя и пароль, а также
адреса серверов входящей и исходящей почты.

image::images/kmail-account.png[Мастер учётных записей]

Вы можете использовать один почтовый клиент для работы с несколькими почтовыми
ящиками. Для каждого из них нужно создать свою учётную запись, вызвав мастер
учётных записей в меню *Сервис -> Мастер учётной записи...*. Редактировать
параметры учётных записей и многие другие характеристики почтового клиента
можно в центре настроек: меню *Настройка -> Настроить KMail...*.

image::images/kmail-settings.png[Настройка KMail]

[float]
==== Использование почтового клиента ====

Работа с почтой осуществляется при помощи кнопок на панели инструментов в окне
программы. Вы можете создавать письма, проверять почтовый ящик на наличие новых
непрочитанных сообщений, пересылать письма и отвечать на них и т.д.

Для создания письма нажмите кнопку *Новое*. В открывшемся окне редактора писем
введите адрес получателя, тему и текст письма.

image::images/kmail-message.png[Редактор писем]

В виде вложения к письму можно пересылать электронные документы, изображения,
архивы и т.п. Для того чтобы добавить вложение, нажмите *Вложить* и выберите нужный файл в
открывшемся окне.

Завершив составление письма, нажмите *Отправить*.

Более подробную информацию об использовании и настройке KMail можно получить,
обратившись к справке при помощи клавиши *F1* или пункта меню *Справка ->
Руководство пользователя KMail*.
