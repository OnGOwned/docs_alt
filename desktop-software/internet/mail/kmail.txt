=== KMail  ===

В качестве программы для  создания, отправки и получения электронных писем по
умолчанию используется почтовый клиент KMail. Ссылку на него можно найти в
группе «Интернет» меню запуска приложений (пункт «Почтовый клиент»).

image::images/kmail.png[Почтовый клиент KMail]

==== Первоначальная настройка KMail ====

Чтобы начать оправлять и получать почту, нужно произвести первоначальную
настройку почтового клиента. При первом запуске KMail это можно сделать с
помощью мастера учётных записей. Последовательно проходя шаги мастера, выберите
тип почтового ящика и введите в предложенные поля сведения об учётной записи
(ваше имя и адрес почтового ящика), её имя и пароль, а также адреса серверов
входящей и исходящей почты. Эти данные нужно предварительно получить у
администратора сети или найти в инструкции по настройке почтового клиента для
почтовой службы, в которой зарегистрирован ваш почтовый ящик.

image::images/kmail-account.png[Мастер учётных записей]

Вы можете использовать один почтовый клиент для работы с несколькими почтовыми
ящиками.  Для каждого из них нужно создать свою учётную запись, вызвав мастер
учётных записей в меню «Сервис» -- «Account wizard...». Редактировать параметры
учётных записей и многие другие характеристики почтового клиента можно в центре
настроек: меню «Настройка» -- «Настроить KMail...».

image::images/kmail-settings.png[Настройка KMail]

==== Использование почтового клиента ====

Работа с почтой осуществляется при помощи кнопок на панели инструментов в окне
программы. Вы можете создавать письма, проверять почтовый ящик на наличие новых
непрочитанных сообщений, пересылать письма и отвечать на них и т.д.

Для создания письма нажмите кнопку «New». В открывшемся окне редактора писем
введите адрес получателя, тему и текст письма.

image::images/kmail-message.png[Редактор писем]

В виде вложения к письму можно пересылать электронные документы, изображения,
архивы. Чтобы добавить вложение, нажмите «Вложение» и выберите нужный файл в
открывшемся окне.

Завершив составление письма, нажмите «Отправить».

Более подробную информацию об использовании и настройке KMail можно получить,
обратившись к справке при помощи клавиши +F1+ или пункта меню «Настройки» --
«Руководство пользователя KMail».
