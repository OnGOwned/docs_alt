=== Evolution ===

Evolution -- графическая клиентская программа управления электронной почтой,
контактами и временем.

image::images/evolution.png[Почтовый клиент Evolution]

[float]
==== Первоначальная настройка ====

Чтобы начать отправлять и получать почту, нужно произвести первоначальную
настройку почтового клиента. При первом запуске Evolution это можно сделать с
помощью мастера установки Evolution. Последовательно проходя шаги мастера,
укажите сведения об учётной записи (ваше имя и адрес почтового ящика), выберите
тип сервера и укажите адреса серверов входящей и исходящей почты.

image::images/evolution-server-types.png[Типы поддерживаемых учётных записей]

Редактировать параметры учётных записей и многие другие характеристики
почтового клиента можно в параметрах Evolution: меню *Правка -> Параметры*.

//application path
Вы можете использовать один почтовый клиент для работы с несколькими почтовыми
ящиками. Для каждого из них нужно создать свою учётную запись, вызвав мастер
учётных записей. Для этого в параметрах Evolution в разделе *Учётные записи*
выберите *Добавить*.

image::images/evolution-settings.png[Настройка Evolution]

[float]
==== Использование почтового клиента ====

Работа с почтой осуществляется при помощи кнопок на панели инструментов в окне
программы. Вы можете создавать письма, проверять почтовый ящик на наличие новых
непрочитанных сообщений, пересылать письма и отвечать на них и т.п.

Для создания письма нажмите кнопку *Создать*. В открывшемся окне редактора писем
введите адрес получателя, тему и текст письма.

image::images/evolution-message.png[Редактор писем]

В виде вложения к письму можно пересылать электронные документы, изображения,
архивы. Чтобы добавить вложение, нажмите *Добавить вложение...* и выберите
нужный файл в открывшемся окне.

Завершив составление письма, нажмите *Отправить*.

//application path
Более подробную информацию об использовании и настройке Evolution можно получить,
обратившись к справке при помощи клавиши *F1* или пункта меню *Справка ->
Содержание*.
