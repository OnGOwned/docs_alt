=== Mozilla Firefox ===

Веб-браузер Mozilla Firefox (произносится «мозилла фаерфокс»), как и другие
приложения, запускается из меню запуска приложений. Веб-браузер предлагает
удобные средства для просмотра гипертекстовых страниц: навигация по вкладкам,
сохранение страниц в закладках, поиск по ресурсам Интернета и ключевым словам
на странице. Новые функции можно добавлять при помощи расширений.

image::images/firefox.png[Веб-браузер Mozilla Firefox]

[float]
==== Просмотр веб-страниц ====

Для того чтобы открыть интернет-страницу, введите её адрес в адресной строке
браузера и нажмите *Enter*. Просматривать несколько страниц одновременно можно
в разных вкладках в одном окне Firefox. Чтобы открыть ссылку в новой вкладке,
нажмите на ней средней кнопкой (колесом) мыши. Для выполнения действий с
вкладкой, вызовите на ней контекстное меню -- при помощи него можно создать
новую вкладку, обновить, закрыть или добавить в закладки одну или все открытые
вкладки. Можно изменять порядок вкладок на панели, перетаскивая их в нужное
положение.

//application path
Страницы часто посещаемых веб-сайтов можно сохранять в закладках. Это можно
сделать разными способами, например, в меню Firefox *Закладки -> Добавить страницу*
или, нажав на изображение звёздочки в правом
краю адресной строки. Организовать закладки по-своему можно при помощи меню
*Firefox -> Закладки -> Показать все закладки*.

[float]
==== Загрузка файлов из Интернета ====

//application path
Вы можете сохранять найденные в Интернете файлы на жёстком
диске вашего компьютера. Текстовые файлы -- в контекстном меню *Сохранить
объект как...*, изображения -- в контекстном меню *Сохранить изображение
как...* и веб-страницы -- меню *Файл -> Сохранить как...*. При сохранении
некоторых файлов откроется диалоговое окно, в
котором нужно подтвердить ваше действие: сохранить файл или
открыть его в предложенной программе.

image::images/firefox-save-dialog.png[Диалог сохранения файла]

//application path
Процесс загрузки и оставшееся время отображаются в окне менеджера загрузок. По
умолчанию все файлы сохраняются в папке загрузок. В настройках
веб-браузера (*Правка -> Настройки*) можно изменить папку для сохранения или
выбрать возможность указывать папку каждый раз при сохранении. Открыть файл или
папку, в которую сохранена загрузка после её завершения, можно из окна загрузок,
используя контекстное меню. Чтобы очистить историю сохранённых файлов, нажмите
*Очистить список*.

image::images/firefox-download-manager.png[Менеджер загрузок]

Кроме того, в окне настроек веб-браузера можно настраивать многие другие
параметры, такие как домашняя страница, работа с вкладками, блокирование
всплывающих окон, личные данные и др. Перемещаться между блоками настроек
можно, нажимая на кнопки вверху окна. Для сохранения изменений нажмите
*Закрыть*.
