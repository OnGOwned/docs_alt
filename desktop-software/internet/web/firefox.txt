=== Mozilla Firefox ===

Веб-браузер Mozilla Firefox (произносится «мозилла фаерфокс»), как и другие
приложения запускается из меню запуска приложений. Веб-браузер предлагает
удобные средства для просмотра гипертекстовых страниц, такие как навигация по
вкладкам, сохранение страниц в закладках, поиск по ресурсам Интернета и
ключевым словам на странице. Новые функции можно добавлять при помощи
расширений.

image::images/firefox.png[Веб-браузер Mozilla Firefox]

==== Просмотр веб-страниц ====

Чтобы открыть интернет-страницу, введите её адрес в адресной строке браузера и
нажмите +Enter+. Просматривать несколько страниц одновременно можно на разных
вкладках в одном окне Firefox. Чтобы открыть ссылку в новой вкладке, нажмите на
ней средней кнопкой (колесом) мыши. Для выполнения действий с вкладкой, вызовите
на ней контекстное меню -- при помощи него можно создать новую вкладку,
обновить, закрыть или добавить в закладки одну или все открытые вкладки. Можно
изменять порядок вкладок на панели, перетаскивая их в нужное положение.

==== Быстрый доступ к нужным сайтам ====

Страницы часто посещаемых веб-сайтов для быстрого доступа к ним можно сохранять
в закладках. Это можно сделать разными способами, например в меню «Закладки» ->
«Добавить страницу в закладки» или нажав на изображение звездочки в правом краю
адресной строки. Организовать закладки по-своему можно при помощи меню
«Закладки» -> «Управление закладками».

==== Загрузка файлов из Интернета ====

Вы можете сохранять найденные в Интернете файлы (в контекстном меню «Сохранить
объект как...»), изображения (в контекстном меню «Сохранить изображение
как...») или веб-страницы (меню «Файл» -> «Сохранить как...») на жёстком
диске вашего компьютера. При сохранении некоторых файлов откроется диалоговое окно, в
котором нужно подтвердить, что вы хотите с ним сделать: сохранить файл или
открыть его в предложенной программе.

image::images/firefox-save-dialog.png[Диалог сохранения файла]

Процесс загрузки и оставшееся время отображаются в окне менеджера загрузок. По
умолчанию все файлы сохраняются в папке рабочего стола. В настройках
веб-браузера («Правка» -> «Настройки») можно изменить папку для сохранения или
выбрать возможность указывать папку каждый раз при сохранении. По завершении
загрузки открыть файл или папку, в которую он сохранён, можно прямо из окна
загрузок, используя контекстное меню. Чтобы очистить историю сохранённых
файлов, нажмите «Очистить список».

image::images/firefox-download-manager.png[Менеджер загрузок]

Кроме того, в окне настроек веб-браузера можно настраивать многие другие
параметры, такие как домашняя страница, работа с вкладками, блокирование
всплывающих окон, личные данные и др. Перемещаться между блоками настроек
можно, нажимая на кнопки вверху окна. Для сохранения изменений нажмите
«Закрыть».
